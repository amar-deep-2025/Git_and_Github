# Git commands and Uses

## Set username
``` git config  --global user.name "username"```  
## Set email
```git config --global user.email "email"```  

## check configurations
``` git config```

## create a new folder
``` mkdir name```
## create a file  
``` touch name.md ```
##  checking your changes 
``` git status```
##  Start tacking files in staging area
``` git add <file-name> 
OR 
git add . 
all files```
